This flake manages five NixOS configurations across four physical or virtual machines. Each host composes a set of [NixOS profiles](/structure) that layer system-wide behaviour — from boot and networking to desktop applications and server services.

## Profile composition

Every host imports at least the `base` profile, which provides the Nix toolchain, user management, SSH, and shell configuration. Bare-metal hosts additionally import `baremetal` for boot, networking, display, and ZFS support. Desktops and laptops layer `userspace` for GNOME, audio, and application packages. The home server uses `server` for self-hosted services, and WSL hosts use the `wsl` profile instead of `baremetal`.

```
base          ← all hosts
baremetal     ← physical machines (hlebak, hydromechanizator, dreamweaver)
userspace     ← interactive desktops (hlebak, hydromechanizator)
server        ← self-hosted services (dreamweaver)
wsl           ← Windows Subsystem for Linux (hlebak-wsl, spycharkozrywarka-wsl)
```
