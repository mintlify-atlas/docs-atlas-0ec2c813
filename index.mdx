This is a personal NixOS configuration built with [Nix Flakes](https://nixos.wiki/wiki/Flakes) and the [blueprint](https://github.com/numtide/blueprint) framework. It manages several machines — a desktop workstation, a laptop, a home server, and WSL environments — through a single, composable repository.

Configuration is fully declarative: from system-wide [Catppuccin Mocha](https://github.com/catppuccin/catppuccin) theming via [Stylix](https://github.com/nix-community/stylix), to per-user dotfiles via [Home Manager](https://github.com/nix-community/home-manager), to encrypted secrets via [agenix](https://github.com/ryantm/agenix).
